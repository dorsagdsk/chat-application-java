# Java Messenger Application

## 📌 Project Description
This is a graphical messaging application developed in Java.  
It simulates a real-world chat system with features like authentication, friend management, direct messaging, and group chats.

## 🚀 Features

### 👤 User System
- User registration and login
- Email verification with 4-digit random code
- Login authentication with error handling

### 👥 Friend System
- Send friend requests
- Accept/reject requests
- Search users by username
- Block/unblock users

### 💬 Direct Messaging
- Send and receive private messages
- Edit and delete messages
- Unsend message within 10 seconds
- Message history view

### 👨‍👩‍👧 Group Chat
- Create groups
- Generate invite links (starting with @)
- Join groups via link
- Send, edit, delete group messages
- Admin can kick members

### 🔐 Security Features
- Blocked users cannot send messages
- Email verification during signup
- Exception handling (UserNotFound, etc.)

### ⏱ Advanced Features
- Message timestamp (system date & time)
- Multithreading for message handling
- Keyboard shortcuts support
- Clear chat history option
